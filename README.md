## Default React + Next.js + Typescript + Styled Components project structure
---

## Tools
* React
* Next.js
* Typescript
* Styled Components

## Clone and Run
```bash
# clone the project
git clone https://github.com/alvescleiton/react-nextjs-typescript-structure.git
```

```bash
# install dependencies
yarn
```

```bash
# run the project
yarn dev
```
